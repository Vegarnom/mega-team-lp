Quick Installation

Get up and running in under a minute. Clone the repository, configure your settings, and start using powerful slash commands like /fix, /feat, and /review.

1

Install Dependencies

brew install ripgrep jq fzf # macOS
2

Clone Repository

git clone https://github.com/Vegarnom/mega-team.git ~/.claude
3

Configure Settings

cp ~/.claude/example.settings.json ~/.claude/settings.json
4

Add PATH for Profiles

echo 'export PATH="$HOME/.claude/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc
5

Start Claude Code

claude
Terminal
$ claude-architect
# Planning & Design mode (opus)

$ claude-builder
# Implementation mode (opus)

$ claude-reviewer
# Code Review mode

$ claude-qa
# Testing & QA mode (haiku)

alias claude-update="cd ~/.claude && git pull && cd -"
